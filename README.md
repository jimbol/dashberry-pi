# Dashberry Pi
Simple dashboard for Raspberry Pi based on [ozua's tutorial](http://www.instructables.com/id/Raspberry-Pi-Wall-Mounted-Calender-and-Notificatio/?ALLSTEPS)

SAMPLE ACTIONS
Chores
- get chores
  - fields
    - title
    - description
    - users
    - asignee
  - filters
    - user
    - get all
- update chore
- create chore
- delete chore
